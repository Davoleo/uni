\section{Task 1}
\makeatletter\def\@currentlabel{Task 1}\makeatother\label{sec:task1}

\subsection{Overview}
\begin{frame}{Overview}
  \begin{block}{Prompt}
    Classification of a dataset with 100 classes of different sports, on the clean test set.
  \end{block}
  \vspace{\baselineskip}
  The sheer number of classes and different variants of sports required us to make use of moderate augmnentation even in this task,
  to generate enough data samples to allow the model to learn correctly while trying to avoid overfitting and class confusion.
\end{frame}

\subsection{Baseline 1}
\begin{frame}[fragile]{Baseline \textendash\ Iteration 0}
  The network we used initially was a CNN with 4 layers.

  \begin{verbatim}
Conv2d: 1-1                            896
MaxPool2d: 1-2                         --
Conv2d: 1-3                            18,496
MaxPool2d: 1-4                         --
Conv2d: 1-5                            73,856
MaxPool2d: 1-6                         --
Conv2d: 1-7                            295,168
MaxPool2d: 1-8                         --
Linear: 1-9                            3,686,500
  \end{verbatim}
\end{frame}

\begin{frame}{Baseline \textendash\ Iteration 0}{Results}
  \textbf{Serious overfitting}: the model wasn't regularized at all and only managed to learn the training set.
  \begin{columns}
    \begin{column}{0.3\linewidth}
      Test Set results: \textit{Accuracy}: 0.458, \textit{F1-Score}: 0.446.
    \end{column}
    \begin{column}{0.7\linewidth}
      \begin{center}
        \includegraphics[width=\textwidth]{../models/v1.1-baseline.png}
      \end{center}
    \end{column}
  \end{columns}
  \let\thefootnote\relax\footnotetext{Trained on RX9070 \textendash\ 6mins 12secs}
\end{frame}

\begin{frame}[fragile]{Baseline \textendash\ Iteration 1}
  In order to tackle overfitting we initially added \textbf{Dropout} to the model:

  \begin{verbatim}
if (self.dropout):
    x = nn.functional.dropout(x, 0.5)
  \end{verbatim}

  This lowered validation accuracy even more, but it also regularized training accuracy to be in line with it, 
  so it was overall a good iteration.
\end{frame}

\begin{frame}[fragile]{Baseline \textendash\ Iteration 2}
  For the 3rd iteration we decided to experiment with \textbf{Batch Normalization} applied on all convolution layers.
  This allows us to normalize the values of the feature extracted by the previous convolutional layer, 
  with additional trainable parameters to allow it to adapt during the training.

  \begin{Verbatim}[fontsize=\small]
Conv2d: 1-1                            896
BatchNorm2d: 1-2                       64
MaxPool2d: 1-3                         --
Conv2d: 1-4                            18,496
BatchNorm2d: 1-5                       128
MaxPool2d: 1-6                         --
Conv2d: 1-7                            73,856
BatchNorm2d: 1-8                       256
MaxPool2d: 1-9                         --
Conv2d: 1-10                           295,168
BatchNorm2d: 1-11                      512
MaxPool2d: 1-12                        --
Linear: 1-13                           3,686,500
  \end{Verbatim}
\end{frame}

\begin{frame}{Baseline \textendash\ Iterations 1 and 2}{Results}
  Dropout resulted in strong regularization, but affected training and validation too much,
  so we decided to keep BatchNormalization effect which was milder and expand from there. 
  \vspace{\baselineskip}
  \begin{columns}

      \textbf{Iteration 1}:
      \begin{itemize}
        \item [train] Loss: 1.1266,\\ Accuracy: 0.6865
        \item [val] Loss: 2.4384,\\ Accuracy: 0.4420 \\ (best acc.: 0.452000)
        \item [test] \textit{Accuracy}: 0.45,\\ \textit{F1-Score}: 0.434.
      \end{itemize}
    \end{column}
    \begin{column}{0.5\linewidth}
      \textbf{Iteration 2}:
      \begin{itemize}
        \item [train] Loss: 0.2143,\\ Accuracy: 0.9328
        \item [val] Loss: 2.8431,\\ Accuracy: 0.4880 \\  (best acc.: 0.488000)
        \item [test] \textit{Accuracy}: 0.52,\\ \textit{F1-Score}: 0.511.
      \end{itemize}
    \end{column}
  \end{columns}
  \let\thefootnote\relax\footnotetext{Trained on RX9070 \textendash\ 6mins 14secs / 7mins 5secs}
\end{frame}

\begin{frame}{Baseline \textendash\ Iteration 4}
  In the 4th iteration we tried adding some \textbf{Gaussian Blur} to simulate
  blur / low quality in the original photos.
  
  \begin{center}
    \includegraphics[width=\linewidth]{img/gaussian_blur.png}
  \end{center}

  This, even if it was applied very lightly, had a bad effect on the network learning features, 
  due to what we assume is, the \textbf{loss of important edges} that defined objects inside the sport scene to aid classification.
\end{frame}

\begin{frame}{Baseline \textendash\ Iteration 4}{Results}
  \textbf{Gaussian Blur} was unnecessary, so we decided to \textit{revert this augmentation}.

\end{frame}

\begin{frame}[fragile]{Baseline \textendash\ Iteration 5}
  During 5th iteration we reintroduced \textbf{Dropout} fixing the oversight we made in Iteration 1:
  \textit{dropout should only be applied during training and not during inference}, 
  and it should be properly guarded in \texttt{pytorch}.
  
  \vspace{\baselineskip}

  \begin{Verbatim}[commandchars=\\\{\}]
if (self.dropout \textcolor{teal}{and self.training}):
    x = nn.functional.dropout(x, 0.5)
  \end{Verbatim}

  \vspace{\baselineskip}

  This fixed some of the \textit{``overfitting''} previously causing performace degradation of the network during inference.
  Since random neurons \textit{should not} be turned off during evaluation.
\end{frame}

\begin{frame}[fragile]{Baseline \textendash\ Iteration 6}
  \textbf{2D Adaptive Average Pooling} helped to regularize the model even more effectively
  removing any remaining trace of overfitting in our model.

  \vspace{\baselineskip}

  This, when prepended to a normal \textit{Flattening} layer, 
  allows the network to extract the best feature match from the Convolution output into the classifier.

  \vspace{\baselineskip}

  \begin{verbatim}
    self.AdaptiveAvgPool2d = nn.AdaptiveAvgPool2d(1)
  \end{verbatim}
\end{frame}

\begin{frame}{Baseline \textendash\ Iteration 6}{Results}

        \item [test] \textit{Accuracy}: 0.454,\\ \textit{F1-Score}: 0.416.
