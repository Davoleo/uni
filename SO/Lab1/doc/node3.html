Visualizzate i file della home directory ordinati in base alla dimensione.

Per visualizzare I file nella cartella home in base alla dimensione ho utilizzato il comando ls;
Le rispettive flag che ho usato sono:
  • -l — per mostrare più informazioni sui file tra le quali la dimensione dei file
  • -S — per ordinare i file per dimensione dal più grande al più piccolo
  • -r — per invertire l'ordine di stampa dei file

Il Comando ls -Sl ha restituito il seguente output: 

    -rw-r--r-- 1 UNIPR\l.davoli9 UNIPR\domain^users 603 nov 18 12:17 big_file.txt
    -rw-r--r-- 1 UNIPR\l.davoli9 UNIPR\domain^users 112 nov 18 12:24 medium_file.txt
    -rw-r--r-- 1 UNIPR\l.davoli9 UNIPR\domain^users  14 nov 18 12:16 small_file.txt
mentre il comando ls -rSl ha restituito il seguente output: 
    -rw-r--r-- 1 UNIPR\l.davoli9 UNIPR\domain^users  14 nov 18 12:16 small_file.txt
    -rw-r--r-- 1 UNIPR\l.davoli9 UNIPR\domain^users 112 nov 18  2021 medium_file.txt
    -rw-r--r-- 1 UNIPR\l.davoli9 UNIPR\domain^users 603 nov 18 12:17 big_file.txt
